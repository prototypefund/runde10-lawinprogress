# Pre-Law-Viewer

Das Online-Tool "Pre-Law-Viewer" soll Gesetzentwürfe und deren Konsequenzen für bestehende Gesetze durch automatische Erstellung einer Synopse sichtbar und nachvollziehbar zu machen.
